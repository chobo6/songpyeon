# 배경음악(BGM) 설계

## 배경 / 목적

원본 게임(`연습.apk`)에서 추출한 배경음악을 실제 대전 진행 중에 재생한다.

## 에셋

`연습.apk`(res/gw.mp3, 2분 22초 — 나머지 13개 mp3는 전부 3초 미만이라 버튼음/효과음으로 판단, 이번
스코프 아님)에서 추출해서 `client/public/game-assets/audio/bgm.mp3`로 옮겨둠. 기존 이미지 에셋과
동일하게 저작권 있는 리소스라 `client/.gitignore`에 이미 포함된 `public/game-assets/` 규칙으로
git에는 안 올라감.

## 재생 규칙

- **재생 시점**: `room.state.phase === "playing"`일 때만 — 로비/대기실/방 목록 화면에서는 재생 안 함.
- **끊김 없이 유지**: 내 차례(`MyTurnScreen`) ↔ 관전(`SpectatorScreen`) 전환은 턴마다(짧으면 4초)
  발생하는데, 그때마다 BGM이 재시작되면 안 됨 — `Game.tsx`에서 두 화면 컴포넌트와 형제 위치에
  `<BgmPlayer>`를 두어, phase가 "playing"으로 유지되는 동안은 이 컴포넌트가 리마운트되지 않게 한다.
- **루프**: 끝까지 재생되면 처음부터 자동 반복.
- **볼륨**: 기본 40%, 별도 조절 UI 없음(사용자 결정).
- **음소거/볼륨 컨트롤**: 이번 스코프에 없음 — 나중에 필요해지면 별도 논의.
- **자동재생**: 게임 시작 시점엔 이미 여러 차례 클릭을 거친 뒤라 브라우저의 사용자 상호작용 요구
  정책에 문제없음. 별도 처리 불필요.
